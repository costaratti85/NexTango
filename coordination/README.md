# Coordination

Carpeta de coordinacion entre Nova, Codex y agentes.

- inbox/: tareas listas para tomar.
- reports/: reportes de ejecucion.
- blocked/: bloqueos que requieren decision humana o de Nova.
- decisions/: decisiones arquitectonicas aceptadas.

No usar esta carpeta como chat infinito. Cada archivo debe ser accionable.
