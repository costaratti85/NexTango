# Project North Star

SistemaIndustrial no reemplaza herramientas buenas: las integra.

Stack:
- ERPNext/Frappe: operación.
- Tango Gestión: fiscal, contable, artículos, clientes, proveedores y precios publicados.
- Excel: pricing humano.
- OCR: ingreso de facturas de proveedor.
- CypCut: nesting.
- Postprocesador propio: leads, secuencia y G-code.
- SistemaIndustrial: lógica industrial, integración y trazabilidad.
