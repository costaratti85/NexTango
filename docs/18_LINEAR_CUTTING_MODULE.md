# Linear Cutting Module

Status: future explicit module.

The existing 1D cutting optimizer remains valuable and must become a SistemaIndustrial module.

Scope:
- profiles, pipes, bars and linear material.
- whole bars and loose remnants.
- cut requests from orders.
- reuse of remnants.
- output: cut plan, consumed bars, generated remnants and stock adjustments.

This module must not be hidden under generic nesting/CAM work. It is a first-class industrial capability.
