# Client Piece Library

Status: future explicit module.

SistemaIndustrial must allow saving recurrent customer pieces.

Purpose:
- avoid redrawing repeated work.
- preserve customer-specific geometry and material preferences.
- enable faster quotes and repeat orders.

Minimum future fields:
- customer
- piece code/name
- source DXF or preset parameters
- material and thickness
- revision/version
- last quoted / last produced references
