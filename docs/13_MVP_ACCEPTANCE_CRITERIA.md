# MVP acceptance criteria

## MVP 0.1 succeeds when

1. A panel decorative preset can calculate chapa and laser meters.
2. That preset can produce an ERPNext Quotation payload.
3. The same preset can create a pending cut part.
4. Pending cut parts can be filtered by material/thickness.
5. A pantografista can generate one ordered DXF for one thickness.
6. The DXF can be opened/imported into CypCut for nesting.

## Out of scope

- Automatic fiscal invoicing.
- Live Tango order import.
- Production scheduling.
- Own nesting.
- Own G-code inside ERPNext.
